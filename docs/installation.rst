============
Installation
============


Stable release
--------------

To install pyside6-plotly, run this command in your terminal:

.. code-block:: console

    $ pip install pyside6_plotly

This is the preferred method to install pyside6-plotly, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for pyside6-plotly can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/bmaranville/pyside6_plotly

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/bmaranville/pyside6_plotly/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/bmaranville/pyside6_plotly
.. _tarball: https://github.com/bmaranville/pyside6_plotly/tarball/master
